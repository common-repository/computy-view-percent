== Description ==
The plugin does not require any settings and works automatically for standard WordPress posts.
 When scrolling the page, the red bar at the top increases.

 == Testing ==
 You can test the plugin on [**this page**](https://demo.tastewp.com/computy-view-percent)

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress
plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Plugin Name screen to configure the plugin
